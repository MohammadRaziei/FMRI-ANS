### `mvap2` installation :

There is three ways:  

##### Classics:  

```bash
sudo apt install python-mvpa2
```

##### PyMVPA2:  

``` bash
conda create -n mvpa2 python=3.6
conda activate mvpa2
pip install numpy==1.14.6
pip install pymvpa2
pip install duecredit
pip install nibabel==2.5.2
```
